Counterfactual Interventions at a Glance

For each patient, the digital twin simulates four evidence-based interventions and ranks them by causal effect size — giving clinicians actionable, personalized prevention plans.

  • ✓ HbA1c reduction — Xue et al. 2019 (HR 1.18)
  • ✓ BP management — SPRINT-MIND 2019 (19% RR)
  • ✓ Physical activity increase — Livingston 2020
  • ✓ LDL cholesterol reduction — Zhu et al. 2021
  • ✓ E-value sensitivity analysis for each effect

Edge AI

Designed for Rural India

The entire causal inference stack runs on a $35 Raspberry Pi 4. No cloud dependency, no internet required. Deploy in any PHC across India with a ₹800 BP cuff and standard blood tests.

  • ✓ 108KB quantized model — fits on microSD
  • ✓ Offline-first architecture
  • ✓ Touch-optimized 7" display interface
  • ✓ FHIR-compatible data export
